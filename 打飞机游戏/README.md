## 检测敌机的碰撞
  1. 每一个敌机都是运动的，每一颗子弹也是运动的 --> 在每一驾敌机运动时,检测和所有游戏界面内存在的子弹的碰撞
  2. 创建全局变量: 把当前游戏界面内存在的所有的子弹放入到该变量里
  3. 敌机运动的过程中, 检测和每一颗子弹的碰撞
  4. 子弹和敌机发生碰撞时,子弹消失
    4.1 把该子弹元素删除掉
    4.2 把该子弹元素从全局变量集合里删除
  5. 敌机和子弹发生碰撞时,敌机血量减少 --> 每颗子弹消耗的血量是100
